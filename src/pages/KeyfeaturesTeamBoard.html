Key Features

Create task

Click the + button on top of a column to add a task with that group by value. For more detail, check 2.4.1 Create a Task

Edit task

User can select any task to see the task form and edit the task (2.4.2). There are alternative ways to quickly edit a field:

Assign: Click the assign icon to assign task.

Status: Click on the status icon ( or or or ) to change the task status

Priority: Click on the priority icon ( or or ) to change the task priority

Load more

By default, each bucket only shows 10 tasks. Click "Load more" to load more tasks in that bucket.

Show Completed tasks

By default, Board View does not show completed tasks (except when grouped by Status). Click "Show Completed" at the bottom of the bucket to show them.

Drag and Drop

Drag a task from one bucket to another to change its value (depending on current group by setting). For example, if currently grouped by Priority, dragging a task from the “Low” priority bucket to “High” will in effect change the Priority value of the task from Low to High.

Move bucket

Click the bucket menu and select to move left/right.

Search

This function behaves similar to the Basic Search function in Grid View.

Filter

This function behaves similar to the Filter function in Grid View.

Group by

Similar to Group by in Grid View, except when user pick Assign To,

they can choose which user they want to see
