Key Features

Chart Drill Down

Chart Drill Down is a function within Team Progress, it provides a grid/list of tasks based on chosen status on Charts.

From Team Progress, wherever there is a chart, Chart Drill Down is clickable. Click on any part of a Chart to show the list of tasks that has the chosen status. In the example below: Choose Status = “Not Started”
