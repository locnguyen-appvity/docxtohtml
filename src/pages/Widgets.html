Widgets

Sticky Note

  • Projects: Show the total number of projects of that PM is working on.
  • Total tasks: Show the total number of tasks of these projects have in Time Frame.
  • Members: Show the total number of members of the projects except those who don’t have any task during the Time Frame. If a member is in multiple projects, it is still counted as 1.
  • Overdue: Show the total number of tasks that have status = “Overdue” of these projects. This will include task that has due date with:
    • Status = “Completed”: Actual completed date > Due Date (later).
    • Other statuses: Current Date > Due Date
  • In-Progress: Show the total number of tasks that have status = “In-Progress” of these projects.
  • Not Started: Show the total number of tasks that have status = “Not Started” of these projects.
  • Type: None.

Status

  • Show percentages of tasks sorted by status within Time Frame.

Priority

  • Show percentages of tasks sorted by priority within Time Frame.

Monthly Workload

  • Shows the chart of tasks by Projects in Monthly. Each bar represents a project.
  • Status filter:
    • All
    • Not Started
    • In-Progress
    • Hold
    • Completed
    • Overdue
  • If Global filter enable or disable a status > In workload filter will have the same available statues as Global Filter.
  • If Global Filter disable a status being chosen in workload filter, by default, workload filter will choose option “All”.
  • Type: Column, Bar, Line.

Stacked Team member

  • Breakdown all tasks for each member base on status (with overdue)
  • User can click on element of the chart to show detail information grouped by projects
  • Type: Stack.

Portrait Team member

  • Breakdown all tasks for each member filtered by status
    • User can click on element of the chart to show detail information grouped by projects
    • Type: Portrait.
