Customization

OverView

When users first access the eTask tab, the default view is Team Tasks. The default columns are:

  • ID
  • Title
  • Assigned To
  • Due Date
  • Status
  • Source

Note that if the eTask window doesn’t have enough space to display all columns (such as on a mobile phone screen), some columns will not show. The column priority order is set by the system.

Create

Apart from the Default views, users can create their own views:

  • Open Left Navigation.

  • Click on the sign, then select the desired view.

  • In the Settings page, enter the view name, and set the rest of the view’s settings. Note that Display Columns supports drag and drop for ordering easily.

  • Configure filter setting as needed.
  • Click “Create” to save the view.

Edit

To edit an existing view, select the view in the display:

Then click on the Setting (wheel icon) to open settings. Click "Save" to save the settings when done.

Set as default

To set a private view to default Task View or Board View, select “Set as default”:

Share/Unshare

When user creates a view, it’s saved for their personal use. You can share a view so other members of the team can use the view.

Otherwise, user can make it private again by clicking on “Recall”.

Clone

Users can clone an existing view to create a new view based on the settings of that view

  • A confirmation message appears

Delete

To delete a View, select “Delete”.

Hide/Show

To hide a view, select “Hide”.

User can access the hidden views. To see them appear on Private Task View again, user can also select “Show”.
