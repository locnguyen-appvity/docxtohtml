Customize dashboard

Once the dashboard is created, user can Filter , change the Setting , or go to Edit mode to add/remove the content.

Filter

Choose Projects:

List out all active projects, select the check-boxes inside dropdown to pick which project user want to take into account.

Standardize Color Mapping By:

Choose a primary project to get color mapping of that project.

Note: If user unselect the primary project, Standardize Color Mapping By will automatically pick the first project in the remain selected list in Choose Projects.

Date:

  • Filtered By: User can pick Start Date or Due Date
  • Time Frame: The amount of time that will be take into account
Note:

  • By default,

    Filter By value = Start Date

    Time Frame value = “Recent Months”

    Recent Month includes “Last Month / Current Month / Next Month”

  • Filter By would take the number of task that has Start Date/Due Date during that Time Frame

Priority, Status, Assigned To:

This works similar to eTask grid view

Click Apply to keep the changes or Cancel to revert the change

Settings

Title: Enter the display name of your dashboard for the list here

Layout: There are 4 columns in the Dashboard. User can choose to have them combined into 1 big main column or split into 2 parts. These are the options

Chart Configuration

User can toggle to show or hide chart elements (legend, series label, tooltips) and pick where they should be.

Click Save to keep the changes or Cancel to revert the change

Note:

If user choose a new layout and click Save, a popup will show up warning the user:

“Changing the layout will delete your current section and widgets.

Do user want to save your changes?”

  • Cancel: Turn off the pop-up window.
  • Don’t Save: Do not save changes and keep the widgets
  • Save: Save and delete the widgets

Edit

User can add, edit, delete Section and Widget in this mode.

Once the edit has been made, user can click to save the change or to revert it back.
