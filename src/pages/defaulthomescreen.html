1. Default Home Screen

eTask is a tab application which is fully integrated into Microsoft Teams environment. After eTask is installed and deployed, the administrator can add eTask as a tab in a channel. Hereâ€™s the default screen layout:

By default, eTask offers the Grid View which has the following main components:

  • Command Bar: Hamburger button, Search, Filter, Group By.
  • Tool Bar: Create button, Edit Inline, Bulk Edit, Export, and Refresh.
  • List View: Sort-able column headers, Hide/Show column headers, list out items with action menus.
  • Paginator: Count items within a page, Page number.
