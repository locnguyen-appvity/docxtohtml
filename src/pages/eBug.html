Features

Besides leveraging Teams' collaboration features, practical functionalities for real world usage are the stars of eBug.

Workflow

eTask provides out-of-the-box workflow for task status changes to handle typical business flow when a tasks status changes from one state to another.

Deep Link Conversation

Your team can discuss and start conversation around a single task within the project.

Customizable Views

eTasks provide a table view, a full feature (include drag and drop) Kanban board view,a nd a Chart dashboard view. These views can be customized per user and can be shared within the team.

Alerts

eTask provide alerts for the following conditions:

  • Task is overdue
  • Task is about due
  • Alert template can be customized, and alerts can be configured to send to select-able users/groups
Data Sources

eTasks can aggregate data from the following system: VSTS, JIRA, and Planner. Columns from these system can be mapped into the default schema used by eTasks. Data edited in eTask are pushed back to the source.

User Mapping

eTasks can associate different accounts from various data sources as a single user. This feature allow you to view and update data from single access account in Teams.

Personal Dashboard

User can view all tasks across all data sources, all teams, all channels into one single personal dashboard in Teams.

Security

Users are authenticated and access securely using the same security architecture of Office 365 and Teams.
