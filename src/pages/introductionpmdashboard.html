Introduction

Default View

eTask dashboard can be found within eTask application. Here is the default screen:

By default, eTask Dashboard will show information from active projects in recent months

  • Sticky report
    • Projects: Show the total number of projects of that PM is working on.
    • Total tasks: Show the total number of tasks of these projects have in Time Frame.
    • Members: Show the total number of members of the projects except those who don’t have any task during the Time Frame. If a member is in multiple projects, it is still counted as 1.
    • Overdue: Show the total number of tasks that have status = “Overdue” of these projects. This will include task that has due date with:
      • Status = “Completed”: Actual completed date > Due Date
      • Other statuses: Current Date > Due Date
    • In-Progress: Show the total number of tasks that have status = “In-Progress” of these projects.
    • Not Started: Show the total number of tasks that have status = “Not Started” of these projects.
  • Project list

    Shows the projects with status = “Active” in briefly details. There are columns display the statistic of project:

    • Project Name: Show the name of the project. This will be Project Display name or Project name (if there is no display name)
    • Tasks: Show the total tasks that include all statuses except “Completed” of these projects.
    • Bugs: Show the total bugs that include all statuses except “Closed” of these projects.
  • Team size
    • Team Size chart shows the members of a project that have all tasks statuses in current Time Frame.
    • Total member in Team Size chart will be different compare to Members in Sticky Report. Briefly information above. Because there might be 1 member working on 2-3 projects.
  • Overdue by Priority
    • Show chart of overdue tasks sorted by priority in all projects in current Time Frame. Priority will be sorted from top to bottom: High, Normal, Low.
  • Status
    • Show chart of tasks with all statuses sorted by Status in all project in current Time Frame.
  • Priority
    • Show chart of tasks by Priority in all Projects in current Time Frame.
  • Project workload by Status
    • Shows chart of all tasks that included all statuses but sorted by Projects. Provide a quick glance for user.
    • Vertical axis: Projects.
    • Horizontal axis: Number of tasks.
    • Within the chart: sorted number of tasks followed by current Status.
    • Overdue: overdue tasks from the incomplete tasks.
  • Monthly workload
    • Shows the chart of tasks by Projects in Monthly. Each bar represents a project.
    • Status filter:
      • All
      • Not Started
      • In-Progress
      • Hold
      • Completed
      • Overdue
    • If Global filter enable or disable a status > In workload filter will have the same available statues as Global Filter.
    • If Global Filter disable a status being chosen in workload filter, by default, workload filter will choose option “All”

Dashboard list

Click on hamburger button to show List of Dashboard

User can click on to create a new dashboard.

When user clicks on the … symbol next to the dashboard, they can:

Set as default: The selected dashboard will be shown every time user open eTask Dashboard. User can only select 1 default dashboard at the time. The default dashboard will have icon next to it.

Clone: A copy of the selected dashboard will be created with all of its widgets, settings, and filter. The copy name is “Clone – [Name of original Dashboard]”.

Delete: Remove the selected dashboard from the list.
