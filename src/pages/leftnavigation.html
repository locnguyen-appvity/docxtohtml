1. Left Navigation

Click the hamburger at the top left to open Left Navigation.

1.1 Default Views

There are five default views in eTask:

  • Team Tasks (Task View): Tasks are displayed in a table list format.
  • Team Board (Board View): Tasks are displayed as cards in a multi column board, which supports drag and drop for quick action.
  • Team Progress (Chart view): Tasks are represented in charts.
  • Team Calendar: Tasks are represented in a calendar.
  • Team Gantt: Tasks are represented in the Gantt chart.

If user has eBug installed, there will be two additional views:

  • Team Bugs (Bug View): Bugs are displayed in a table list format.
  • Team Bugs Progress: Bugs are represented in charts.

1.2 Public Views

Public Views are which you can share with other members. But first, you need to create a Private View and then you can share it.

Public Views also show the shared Views from other members. You can only view it but cannot modify the view of others.

1.3 Private Views

Private Views are which you are the only one that can see them. In Private View, you can modify it.
