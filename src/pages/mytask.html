My Task

Overview

Allow user to have a look on all tasks that being assigned to them.

Click on “” on the left menu, then choose “eTask” (if user has already installed eTask) or find an app named “eTask” to get into eTask board.

Alternatively, from the global search of Microsoft Teams, type in “eTask” and select eTask.

Here is the default screen of “My Tasks” tab:

Search

This function behaves similar to the Basic search function in Grid View.

Filter

This function behaves similar to the Filter function in Grid View.

Group by

This function behaves similar to the Group by function in Grid View.

Sync to Outlook

Toggle on to synchronize tasks into Outlook.

Click on “…” at the top-right of “My Tasks” grid view.

Toggle on “Sync to Outlook”.

Tasks from eTask will be synchronized into Outlook. From Outlook, click on list TODO.

Click on “Appvity eTask” to view tasks.

Note:

  • Synchronization between eTask and Microsoft TODO is one-way, synchronizing tasks from eTask to Microsoft TODO only. Any changes in Microsoft TODO will not affect the tasks in eTask.
  • Any changes in eTask will be automatically affected on Microsoft TODO. If there are tasks that are missing on Microsoft TODO, eTask will save the missing tasks in “Logs”, under “Settings” like below.
Toggle off to desynchronize tasks from Outlook.

Toggle off “Sync to Outlook”.

Tasks in TODO list will be removed.
