Why can't I add a picture or insert rich text to the body of a task?

Appvity eTask and VSTS source support image and rich text insertion to the body of a task, while Jira and Planner do not support. Capacity of inserted image can only be 70 KB or lower.

Can I select and edit multiple items?

No, you cannot. eTask supports selecting or editing one at a time.

Once I create a task and associate that with an external data source, like JIRA, can I re-associate that task to a different data source?

No you cannot. To change the task to a different data source, you can “clone” that task to a new one and change the data source.

What will happen if I delete a task that is linked to an external data source? Will the task in the external data source be deleted?

eTask is synchronized with other external data sources, so if you delete a task that is linked to an external data source, that task will be deleted in the external data source too.

If I link eTask to an external data source like Planner, what will happen when a user adds a task using Planner interface?

If you add a task using Planner interface, that task will be shown in eTask too once you reload the eTask tab.

What will happen if I used an attribute such as Status=Hold, and the external data source that this task is linked to does not support that choice?

The status will be changed into Not Started.

Who will receive notification when a task is updated?

The assignee(s), the creator, and project manager. You can set to whom and when the notifications is sent by going to Settings > Others > Email Alert.

Can I create multiple eTask tabs in a Team or Channel?

Yes, you can.

Can external guest users participate in eTask?

No. Only users that are added into Team can use eTask in multiple channels.

Does eTask require an external data source to work? Where is the data store?

No. eTask can work with its default build-in database, which is located on Azure cloud portal.

What happens in eTask when I modify a record in the external data source? For example, modify task name in Planner?

When you modify a record in the external data source, that record in eTask will be changed too.

Why can’t I see anything in Team Board when grouped by “Assigned To”?

Because the assignee's names have not been added manually. Adding the assignee's name will allow you to see the tasks belong to that person.

Can I create a Task View with a specific source only (Task View of tasks from Jira)?

Yes, you can. You can filter Task View by source in the Task View settings panel.

Can I set multiple accounts from external sources into 1 user in Report View?

Yes, you can. Go to Settings > User Mapping to map users from other sources.

How can I embed a single task into Conversation?

In the chat box, type “@eTask” > Find a task (or click on “Messaging extensions”) > Choose eTask > Choose a specific task > Click on the send icon to embed a task into the conversation.

What will happen if I remove an eTask tab? Will all my data be deleted?

No, your data will not be deleted when you remove an eTask tab. Data from other sources will be stored in themselves while Appvity eTask data will be stored in AD. Data from VSTS, Jira, or MS Planner will be deleted unless they are deleted from external sources. Data from Appvity eTask source will be deleted when you remove the channel.

What will happen if I remove the Channel that has my eTask tab? Will all my data in eTask be deleted?

Only data in the Appvity eTask data source will be deleted while data in other data sources will not be.

Why can’t I delete my public views?

You cannot delete your public views or other’s. You can delete your private view only. By that, public view will not be shown.

Why are my settings for “Allow page size” not working?

If one of your private views is set as “Default View”, settings for “Allow page size” in eTask settings will not work unless you change settings in that private view itself.

What file formats are supported when upload attachment to a task?

eTask supports all types of files within 5MB and below.
