Task Form Overview

When you Create, or click on a task, the task form will be displayed, and the following fields are presented (fields with asterisk * are required):

  • Title: Name of the task.
  • Assigned To: Type user name or email to find users to assign this task. Only users from the selected Source are available for selecting.
  • Description: Enter description of the task (plain text if Source is Microsoft Planner).
  • Source: Only show data sources that have been configured for this Channel. Values include:
    • Appvity eTask
    • Azure DevOps – [Project/Area name]
    • JIRA – [Project name]
    • Microsoft Planner – [Plan name]
  • Phase: Phase values depend on the Source selected
    • eTask: Select Phase
    • Azure DevOps: Select Iteration from configured Azure DevOps project.
    • JIRA: Select Sprint from configured JIRA project.
    • Microsoft Planner: Plan will be auto-assigned.
  • Story: Story values also depend on the Source selected
    • eTask: Select Story
    • Azure DevOps: Select User Story from configured Azure DevOps project.
    • JIRA: Select Story from configured JIRA project.
    • Microsoft Planner: Select Bucket.
  • Start Date: Select Start Date & Time.
  • Due Date: Select Due Date & Time.
  • Priority: Select Priority. Default values include: High, Normal, Low.
  • Status: Select Status. Default values include: New, Active, Hold, Completed, Removed.
  • Attachment: Click Add Attachment to attach a file.
  • Related To: Select any Task/Bug related to the current task.

Click "Save" to save the task.

Note:
  • Source: Only source that is set up in Settings (4.3.2) will appear in the list.
  • Phase & Story: The primary values in Settings (4.3.4.1 or 4.3.4.2) will determine the default values of these fields.
  • Start Date & Due Date: User’s preference in Settings (4.3.4.8.1) will determine if these fields are required.
  • The "Save" button is only enabled when all required fields have values.

Create

  • To add a new task, click “Create Task” for Task View.

  • Or “+” for Board View.

Edit

Click on the task title or select "Edit" from the task action menu to open the task form and edit.

Assign

Click on the task action menu and select "Assign". Only users from the source of this task are available for selection.

Clone

Click on the task action menu and select Clone. The content of current task will be copied and displayed in the task form. Saving this form will create a new task.

Get Link

Click on the task action menu and select Get Link. Then click Copy or press Ctrl C on the keyboard to copy a link to this task.

Delete

Click on the task action menu and select "Delete".

User Popup Card

The user popup card displays detailed information about people and provides one-click access to several ways of communicating with someone.

When user hovers the mouse over a user picture, they’ll see:

  • Profile picture.
  • Contact information.
  • Functions:
    • Status (Online, Busy, Away...).
    • Open chat
    • Recent email(s).
    • Start video call.
    • Start audio
    • Task assigned to the owner of the popup card.
